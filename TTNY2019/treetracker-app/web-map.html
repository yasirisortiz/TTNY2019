How it works

The web map is a live view of the existing trees registered with Treetracker. Starting at global level clustering will show the available data points per continent splitting it into country, region or state down to the district level. Web Map 2.0 will show more statistics and will incorporate a more functional mobile version for a more exciting user experience.
